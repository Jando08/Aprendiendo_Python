# 📦 Variables y Tipos de Datos

Las variables son contenedores para guardar información. Python detecta el tipo de dato automáticamente.

## 🏀 Tipos de Datos Básicos
* **Strings (Texto):** `nombre = "Austin Reaves"`
* **Integers (Enteros):** `edad = 22`
* **Floats (Decimales):** `promedio = 25.4`
* **Booleans:** `es_titular = True`

## 📊 Estructuras de Datos Complejas
* **Listas (Colecciones ordenadas):** `premios = ["MVP", "All-Star"]` (Se indexan desde `0`).
* **Diccionarios (Clave: Valor):** 
```python
  puntos_jugadores = {"Curry": 30, "LeBron": 25}
