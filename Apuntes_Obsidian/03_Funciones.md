# ⚙️ Funciones y Scope
Una función es un bloque de código empaquetado que hace una tarea específica y solo se ejecuta cuando lo llamas. 
## 🛠️ Parámetros y Retorno 
* **Parámetros múltiples:** Podemos pasarle varias variables a la función. * **Keyword Arguments:** Invocar la función aclarando qué valor va a qué parámetro (ej: `calcular_puntos(partidos=10, promedio=25)`). * **Return:** Devuelve un valor procesado para guardarlo en una variable fuera de la función.
## 🌐 Scope (Alcance de las Variables) 
* **Variables Locales:** Creadas dentro de una función. Solo viven ahí adentro. * **Variables Globales:** Creadas fuera de las funciones. Se pueden leer desde cualquier parte.
