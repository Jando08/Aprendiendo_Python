# Documentación del Módulo: Ejercicios de Estructuras de Datos (Diccionarios)

Este módulo centraliza una colección de ejercicios prácticos diseñados para el aprendizaje y dominio de las estructuras de datos de tipo diccionario en Python. Los proyectos cubren diversas áreas funcionales, simulando sistemas reales como bibliotecas, tiendas, escuelas y videojuegos, lo que permite aplicar los conceptos aprendidos en contextos variados y prácticos.

## 📚 Estructura de Archivos del Proyecto

Esta sección lista los archivos fuente que componen el módulo, indicando brevemente su propósito.

| Archivo | Tipo | Propósito |
| :--- | :--- | :--- |
| `Ejercicio_Biblioteca.py` | Módulo Python | Implementación de funcionalidades relacionadas con la gestión de una biblioteca (préstamo y reserva de libros). |
