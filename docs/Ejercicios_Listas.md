# Módulo: Ejercicios_Listas

Este módulo recopila una serie de ejercicios prácticos diseñados para reforzar el conocimiento y la manipulación de estructuras de datos de tipo lista (listas) en Python. Su propósito es servir como un repositorio de ejemplos funcionales para tareas comunes de manejo de colecciones.

**Ubicación del proyecto:** `/home/tadeofed/Aprendiendo_Python/Ejercicios_Listas`

---

## Estructura de Archivos (`File Structure`)

Los siguientes archivos contienen la implementación de los ejercicios. Cada archivo está dedicado a demostrar una operación específica sobre listas.

| Archivo | Propósito | Descripción Técnica |
| :--- | :--- | :--- |
| `Ejercicio_Append.py` | Módulo Python | Implementa ejercicios relacionados con la adición de elementos al final de una lista (`.append()`). |
| `Ejercicio_Extend.py` | Módulo Python | Demuestra el proceso de concatenación o extensión de una lista con los elementos de otra lista (`.extend()`). |
| `Ejercicio_Insert.py` | Módulo Python | Contiene ejemplos sobre la inserción de elementos en una posición específica de una lista (`.insert()`). |
| `Ejercicio_Len.py` | Módulo Python | Ejercicios centrados en la determinación del tamaño de una lista (función `len()`) y la manipulación de elementos basándose en su longitud. |
| `Ejercicio_Pop.py` | Módulo Python | Practica la eliminación de elementos por índice o por valor, con énfasis en la recuperación del valor eliminado (`.pop()`). |
| `Ejercicio_Remove.py` | Módulo Python | Contiene ejemplos para la eliminación de elementos de la lista utilizando la sintaxis `del` o `remove()`. |
| `Ejercicio_Sort.py` | Módulo Python | Implementa ejercicios para ordenar listas, tanto de forma ascendente como descendente, utilizando métodos como `.sort()`. |
