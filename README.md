# 🐍 Guía de Documentación del Proyecto: Aprendiendo Python

Este repositorio ha sido diseñado como una herramienta integral de aprendizaje práctico en Python. Su objetivo es guiar al usuario a través de los conceptos fundamentales de la programación, utilizando ejercicios y proyectos temáticos que refuerzan la teoría con la práctica constante.

## 📚 Visión General del Proyecto (Overview)

*   **Tecnología Principal:** Python
*   **Propósito:** Servir como un *roadmap* de aprendizaje progresivo, dividiendo el conocimiento en módulos específicos (variables, estructuras de control, funciones, manejo de archivos, etc.).
*   **Enfoque:** El proyecto está estructurado para facilitar un aprendizaje guiado, y los ejercicios de la práctica son progresivamente más complejos.

### 🏛️ Estructura de Módulos de Aprendizaje

#### 📚 Módulo de Prácticas Fundamentales
Contiene la mayor parte del contenido didáctico y se divide por concepto o dificultad.

*   **`.../Aprendizaje/`**: Carpetas de práctica temática.
    *   **`Bucles y Condicionales`**: Práctica con `for`, `while`, `if/else`.
    *   **`Funciones`**: Creación y llamada de funciones.
    *   **`Clases y POO`**: Práctica con Programación Orientada a Objetos.
*   **`.../Ejemplos/`**: Ejemplos de código reutilizable.
    *   `mi_proyecto.py`
    *   `calculadora.py`
